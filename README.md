# OnlineReservationSystem

## Objective
The OnlineReservationSystem project is a Java application developed using Eclipse IDE. It simulates a reservation system allowing users to book, view, and manage reservations.

## Project Overview
The project includes multiple Java classes handling different parts of the system:

- **ReservationManager.java**  
  Handles core reservation logic including adding, updating, and canceling bookings.

- **UserInterface.java**  
  Provides the graphical interface for interacting with the system.

- **DatabaseHandler.java**  
  Manages data storage, retrieval, and persistence for reservations.

- **Main.java**  
  Entry point of the application that initializes the system and launches the interface.
